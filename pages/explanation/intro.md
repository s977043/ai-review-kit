---

River Reviewer (RR) は、**Upstream → Midstream → Downstream** の流れに沿ってチームのレビュー観点を整理・実行するためのフレームワークです。スキルは YAML フロントマター + Markdown で記述し、`schemas/skill.schema.json` で検証します。

このドキュメントでは以下をカバーします。

- **Explanation**: River Reviewer の設計思想とフロー解説
- **Tutorials**: スキル作成など手を動かす手順
- **How-to**: GitHub Actions 連携やトレーシングなどの実践ガイド
- **Reference**: スキーマや設定のリファレンス

まずはフローの概要から読み進め、必要に応じてチュートリアルやリファレンスを参照してください。
